1、不同路径II的状态转移方程：  
定义状态 dp[i][j] 表示到从坐标 [0][0] 到坐标 [i][j] 位置的不同路径的数量。  
若 grid[i][j] == 1，表示有障碍物不可达，dp[i][j] = 0；  
若 grid[i][j] == 0，表示可达，dp[i][j] = dp[i - 1][j] + dp[i][j - 1]。
